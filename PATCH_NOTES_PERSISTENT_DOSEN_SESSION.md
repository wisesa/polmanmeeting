# Catatan Revisi

Perubahan terbaru:

1. Login dosen dibuat tetap aktif selama dosen belum logout.
2. Istilah teknis pada tampilan aplikasi diganti dengan bahasa yang lebih mudah dipahami.
3. Informasi tentang teknologi yang dipakai tidak lagi ditampilkan pada halaman login, halaman data wajah, halaman presensi, daftar presensi, maupun pesan bantuan.
4. Tombol dan kotak debug kamera tidak ditampilkan pada halaman pengguna.
5. Tampilan data wajah menggunakan istilah sederhana seperti “Data wajah siap”, “Membaca wajah”, dan “Menyiapkan pemeriksaan wajah”.

Catatan:

- Login tetap dapat hilang jika pengguna menghapus data situs, memakai mode pribadi/incognito, atau menekan logout.
- Pastikan data wajah dosen sudah tersimpan sebelum dosen mencoba login.
